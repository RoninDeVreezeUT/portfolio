Creative Technology Module 3

For the third module, we had the opportunity to work with the Eindhoven based client Bureau Moeilijke Dingen to develop an unconventional musical interface.

To inspire users, we decided to make a unconventional but intuitive musical interface, this was doen by using ESP-32's to create multiple identical unit that can measure the distance to the users hand. This data is used to control certain parameters in the synthesizer. Other features like looping and a beat were implemented to allow experienced users to take their musical skills further.
